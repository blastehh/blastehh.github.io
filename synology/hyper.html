# Hyper Backup

# Backing up files from the NAS to Windows Azure

Open up the Hyper Backup interface

Select the + sign in the bottom left to create a new backup task. Select Azure as the destination.

Check on Azure for the correct settings, you can create a new container from this interface.

Select the folders to be backed up.

Set up the backup schedule, enable client-side encryption and generate a password inside KeePass.

Set up the rotation settings

The task will then be created.
